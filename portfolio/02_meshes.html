Project Procedural Meshes

Geometry Generation

Project Overview

This assignment focuses on procedural geometric modeling for planes, cubes, spheres, tori, cylinders, and cones. The goal is to generate reusable mesh data instead of relying on imported models.

Planned Parameters

  • Shape selection for each primitive type.
  • Segment, ring, radius, height, and subdivision controls where appropriate.
  • Normal, UV, and wireframe visualization toggles for debugging generated geometry.

Reflection

Procedural meshes are a useful test bed because they make data layout mistakes visible immediately. This page will document the mesh generation choices and screenshots as the assignment progresses.
