Project Value Noise

Procedural Texture Generation

Project Overview

Value Noise generates procedural texture data by assigning random values to a repeating lattice, hashing input coordinates into that lattice, and interpolating between neighboring samples. This demo supports 1D, 2D, and 3D noise with adjustable period dimensions and smoothing methods.

Tasks Completed

  • Implemented repeatable permutation hashing for wrapped noise coordinates.
  • Built linear, bilinear, and trilinear interpolation for templated value types.
  • Added linear, cosine, smoothstep, and quintic fade controls.
  • Implemented a templated ValueNoise class with editable random values and period resizing.
  • Uploaded generated RGBA data into OpenGL/WebGL textures for live display.
  • Built the Emscripten release and embedded the interactive web demo below.

Reflection

The most useful part of this project was seeing how small interpolation choices change the final texture. Linear smoothing makes the lattice visible, while quintic smoothing produces softer transitions that work better for marble, wood, turbulence, and fractal sum patterns.
